
## To get the results in the paper:

* This section provides results of Figure 1 in the main text, Figures F1 and F2, Figures F15-F17, Figures F24 and F29 in the Supplementary Material.

* Run S1_simulation.R to get the results of Algorithm 1 and Algorithm 2 in our paper, the Knockoff method, the Gaussian Mirror method, and the Gaussian Mirror method modified by FDP+ threshold.

* Run S1_simulation_origin.R to get the results of p-value-based methods with original data.

* Run S1_simulation_nongaussian.R to get the results with non-Gaussian errors.

* Run S1_simulation_ds.R to get the results with data-splitting.

* Put simulation results of S1_simulation.R in the folder S1_plot.

* Put simulation results of S1_simulation_origin.R in the folder S1_plot_origin.

* Put simulation results of S1_simulation_nongaussian.R in the folder S1_plot_nongaussian.

* Put simulation results of S1_simulation_ds.R in the folder S1_plot_ds.

* Run S1_plot.R to generate all subfigures of Figures 1, F1 and F2.

* Run S1_plot_origin.R to generate all subfigures of Figures F15-F17.

* Run S1_plot_nongaussian.R to generate all subfigures of Figure F24.

* Run S1_plot_ds.R to generate all subfigures of Figure F29.

* The folder S1_plot contains all subfigures of Figures 1, F1 and F2. 

* The folder S1_plot_origin contains all subfigures of Figures F15-F17.

* The folder S1_plot_nongaussian contains all subfigures of Figure F24.

* The folder S1_plot_ds contains all subfigures of Figure F29.
