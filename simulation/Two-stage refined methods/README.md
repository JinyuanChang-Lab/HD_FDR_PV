
## To get the results in the paper:

* This section provides results of Figure 3 in the main text, Figures F11, F12 and F26 in the Supplementary Material.

* Run S4_simulation.R to get the results of Algorithm 1 and Algorithm 2 in our paper, the Knockoff method, the Gaussian Mirror method, and the Gaussian Mirror method modified by FDP+ threshold.

* Run S4_simulation_nongaussian.R to get the results with non-Gaussian errors.

* Put simulation results of S4_simulation.R in the folder S4_plot.

* Put simulation results of S4_simulation_nongaussian.R in the folder S4_plot_nongaussian.

* Run S4_plot.R to generate all subfigures of Figures 3, F11 and F12.

* Run S4_plot_nongaussian.R to generate all subfigures of Figure F26.

* The folder S4_plot contains all subfigures of Figures 3, F11 and F12.

* The folder S4_plot_nongaussian contains all subfigures of Figure F26.
