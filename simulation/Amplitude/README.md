
## To get the results in the paper:

* This section provides results of Figure 2 in the main text, Figures F3-F7, F18-F20, F25 and F30 in the Supplementary Material.

* Run S2_simulation.R to get the results of Algorithm 1 and Algorithm 2 in our paper, the Knockoff method and the Gaussian Mirror method, and the Gaussian Mirror method modified by FDP+ threshold.

* Run S2_simulation_origin.R to get the results of p-value-based methods with original data.

* Run S2_simulation_nongaussian.R to get the results with non-Gaussian errors.

* Run S2_simulation_ds.R to get the results with data-splitting.

* Put simulation results of S2_simulation.R in the folder S2_plot.

* Put simulation results of S2_simulation_origin.R in the folder S2_plot_origin.

* Put simulation results of S2_simulation_nongaussian.R in the folder S2_plot_nongaussian.

* Put simulation results of S2_simulation_ds.R in the folder S2_plot_ds.

* Run S2_plot.R to generate all subfigures of Figures 2 and F3-F7.

* Run S2_plot_origin.R to generate all subfigures of Figures F18-F20.

* Run S2_plot_nongaussian.R to generate all subfigures of Figure F25.

* Run S2_plot_ds.R to generate all subfigures of Figure F30.

* The folder S2_plot contains all subfigures of Figures 2 and F3-F7.

* The folder S2_plot_origin contains all subfigures of Figures F18-F20.

* The folder S2_plot_nongaussian contains all subfigures of Figure F25.

* The folder S2_plot_ds contains all subfigures of Figure F30.
