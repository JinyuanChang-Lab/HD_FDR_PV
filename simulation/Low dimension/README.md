
## To get the results in the paper:

* This section provides results of Figure 4 in the main text, Figure F13 and Figure F14 in the Supplementary Material.

* Run S5_simulation.R to get the results of Algorithm 1 and Algorithm 2 in our paper, and the BBH method.

* Put simulation results of S5_simulation.R in the folder S5_plot.

* Run S5_plot.R to generate all subfigures of Figures 4, F13 and F14.

* The folder S5_plot contains all subfigures of Figures 4, F13 and F14.
