
## To get the results in the paper:

* This section provides results of Figures F8-F10, F21-F23 in the Supplementary Material.

* Run S3_simulation.R to get the results of Algorithm 1 and Algorithm 2 in our paper, the Knockoff method, the Gaussian Mirror method, and the Gaussian Mirror method modified by FDP+ threshold.

* Run S3_simulation_origin.R to get the results of p-value-based methods with original data.

* Put simulation results of S3_simulation.R in the folder S3_plot.

* Put simulation results of S3_simulation_origin.R in the folder S3_plot_origin.

* Run S3_plot.R to generate all subfigures of Figures F8-F10.

* Run S3_plot_origin.R to generate all subfigures of Figures F21-F23.

* The folder S3_plot contains all subfigures of Figures F8-F10.

* The folder S3_plot_origin contains all subfigures of Figures F21-F23.
